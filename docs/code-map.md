# Code Map

## Directory Map

| Path | Purpose |
|---|---|
| `decompose.py` | CLI script: JSON preset -> decomposed directory |
| `compose.py` | CLI script: decomposed directory -> JSON preset |
| `teams/` | All team presets (decomposed directories + composed JSONs) |
| `teams/claude-codex-advanced/` | Decomposed 6-agent Development Team preset |
| `teams/claude-codex-advanced.json` | Composed JSON for 6-agent preset |
| `teams/claude-codex-gemini-advanced/` | Decomposed 9-agent Development Team preset |
| `teams/claude-codex-gemini-advanced.json` | Composed JSON for 9-agent preset |
| `teams/requirements-team/` | Decomposed 3-agent Requirements Team preset |
| `teams/requirements-team.json` | Composed JSON for Requirements Team preset |
| `specs-flow-template/` | Template for specs pipeline directory structure and VRD template |
| `drafts/` | Draft content (non-config, e.g. LinkedIn posts) |
| `sop-*.md` | SOP audit/validation/simulation documents (root-level) |
| `.claude.json` | Claude Code project configuration |
| `.devchain/` | Devchain local state |
| `.gitignore` | Ignores `drafts/` |

## Application Entry Points

- `decompose.py` -- `python3 decompose.py <config.json>`
- `compose.py` -- `python3 compose.py <directory>`

## Important Configuration Files

| File | Controls |
|---|---|
| `*/_structure.json` | Key ordering and source map for compose round-trip |
| `*/manifest.json` | Preset metadata (name, version, description, slug) |
| `*/config.json` | Top-level settings (initial prompt, auto-clean statuses, epic assignment template) |
| `*/statuses.json` | Board columns/workflow states (Draft, New, In Progress, Review, Done, Blocked, Backlog, Archive) |

## Notable Scripts

- `decompose.py` -- one command, no flags beyond the input JSON path
- `compose.py` -- one command, no flags beyond the input directory path

## Generated / Build Outputs

- Composed `.json` files at repo root are generated by `compose.py` from their corresponding directories

## Ignored Paths and Rationale

- `drafts/` -- non-config draft content, gitignored
- `.git/`, `.idea/`, `.claude/`, `.gemini/`, `.devchain/` -- tooling/IDE/VCS state
